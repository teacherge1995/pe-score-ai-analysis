# pe-score-ai-analysis
本系统是历下区统一体育测试标准的教学辅助工具，专为体育教师设计，可实现体育成绩的批量处理、智能分析与教学策略生成，解决传统人工统计效率低、分析难的痛点。 系统核心功能： 1. 标准化成绩录入模板下载 2. 批量数据处理与自动成绩核算 3. 班级成绩排名与优良率统计 4. 可视化数据图表生成 5. 基于成绩分析的教学训练策略建议
【登录说明】测试账号：sportsystem，测试密码：sportv2.0
【使用步骤】1. 安装 Python3.8 及以上版本；2. 打开命令行进入项目文件夹，输入 pip install -r requirements.txt 安装依赖；3. 运行 a体育成绩批处理智能分析系统密码版.py，打开浏览器输入http://localhost:5000即可使用，本地局域网使用也可以 http://192.168.69.94:5000
